Database deployment

PHP.Gt's automatic database deployment can only go so far without your input. It seems that there is no database created yet for your application, and new databases can't be created without MySQL root access (that's a good thing though!).

From here, you have a choice:

  1. Enter your MySQL root password here, and let PHP.Gt deploy your database automatically. Click here to understand the security risks!
  2. Input the SQL manually yourself. Clicking the above button will output the SQL to paste into your server.
